# Sentinel-icu-- A SAGE Sepsis(SAGE-SEPSIS ALERT &GUIDANCE ENGINE)
Clinical AI system for early sepsis detection in ICU patients.  BiLSTM + Temporal Attention trained on 91,791 MIMIC-IV patients  (AUROC 0.9781). 17-hour advance warning over SOFA score.  FastAPI backend + Streamlit clinical dashboard.
SentinelICU detects sepsis up to 17 hours before 
clinical recognition - trained on real ICU data from 
91,791 patients.
